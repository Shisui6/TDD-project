### Prerequisites

In order to run this project you need:

```sh

```

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/Shisui6/TDD-project.git
```

### Install

Install this project with:

```sh
  cd TDD-Project
  bundle install
```
